## 0.1.0-alpha.1 - 2026-05-11

Initial public alpha release.

Release posture:

- GitHub tag/release only; PyPI publishing remains out of scope.
- Python package version is `0.1.0a1` to follow PEP 440 prerelease syntax.
- Install from tag with `python -m pip install git+https://github.com/vince-winkintel/hermes-browser-workspace.git@v0.1.0-alpha.1`.

Publicization and packaging:

- Prepared the completed Phase 0-5 Browser Workspace roadmap for public repository publication.
- Added public-readiness repository hygiene, contribution, security, license, and CI artifacts.
- Added package resource copies for starter helper and bundled skill files so wheel installs can bootstrap without relying on a source checkout layout.

Included roadmap outcomes:

- Phase 0 specification, architecture, roadmap, and security model
- Phase 1 local MVP plugin/tool registration and workspace bootstrap
- Phase 2 safer adaptation flows for helper proposals, validation, artifacts, and cleanup
- Phase 3 domain skill maturity, verification/staleness metadata, examples, recipes, and local import/export
- Phase 4 upstream Hermes PR candidate artifact and plugin-local/upstream boundary decisions
- Phase 5 conservative community ecosystem model, contribution guidance, trust/review model, compatibility matrix, and community-domain-skill template
