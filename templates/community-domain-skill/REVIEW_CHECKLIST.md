# Review Checklist

- [ ] Domain scope is clear and appropriate for sharing.
- [ ] No cookies, tokens, localStorage, API keys, credentials, session IDs, private URLs, customer records, or raw private screenshots are present.
- [ ] Examples are synthetic, public, or redacted.
- [ ] Selectors are documented with stability notes.
- [ ] Recipes are generic and free of account-specific assumptions.
- [ ] Metadata includes source repository, source commit, maintainers, compatibility, verification, and known limitations.
- [ ] Any helper code is absent or manually reviewed without execution.
- [ ] The skill does not encourage bypassing authorization, paywalls, terms of service, or anti-bot controls.
- [ ] Import guidance says local copies remain `draft` and `pending_review` until verified.
